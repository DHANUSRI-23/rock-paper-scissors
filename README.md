# Rock Paper Scissors 🎮

A simple and interactive Rock-Paper-Scissors game built using HTML, CSS, and JavaScript.

## 🚀 Features
- User vs Computer gameplay
- Real-time result display
- Score tracking 
- Responsive design 

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
